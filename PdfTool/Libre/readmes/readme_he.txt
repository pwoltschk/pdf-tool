
======================================================================
הקרא אותי של LibreOffice 5.4
======================================================================


לקבלת העדכונים האחרונים לקובץ הסבר זה, ניתן לעיין בכתובת http://www.libreoffice.org/welcome/readme.html

הקובץ מכיל מידע חשוב בנוגע לתכנה LibreOffice. מומלץ לקרוא מידע זה בקפידה בטרם התחלת ההתקנה.

קהילת LibreOffice אחראית לפיתוח המוצר הזה והיא מזמינה אותך לשקול לקחת חלק על ידי חברות בקהילה. למשתמש חדש, מומלץ לבקר באתר של LibreOffice, בו ניתן למצוא מידע רב באשר למיזם LibreOffice והקהילות שסביבו. נא לגשת לכתובת http://www.libreoffice.org/.

האם החבילה LibreOffice אכן חופשית לכל משתמש שהוא?
----------------------------------------------------------------------

חבילת LibreOffice הנה חופשית לשימושו של כל אדם. באפשרותך לקחת את עותק זה של LibreOffice ולהתקין אותו על כמה מחשבים שבא לך ולהשתמש בו לכל מטרה שבא לך (לרבות שימוש מסחרי, ממשלתי, הנהלה ציבורית וחינוכי). לפרטים נוספים ניתן לעיין בטקסט שארוז יחד עם מארז הורדה זה של LibreOffice.

מדוע LibreOffice חופשית לשימושו של כל אדם?
----------------------------------------------------------------------

ניתן להשתמש בעותק זה של LibreOffice ללא תשלום כיוון שתורמים פרטיים ומממנים מסחריים תכננו, תכנתו, בדקו, תרגמו, תיעדו, תמכו, שיווקו וסייעו בדרך רבות ומגוונות להפוך את LibreOffice למה שהיא כיום – חבילת התכנה המובילה בעולם ליישומים משרדיים בקוד פתוח לבית ולמשרד.

אם מאמציהם מוערכים בעיניך וברצונך לוודא כי LibreOffice תוסיף להיות זמינה בעתיד, כדאי לשקול ברצינות לתרום למיזם - ניתן לבקר בכתובת http://www.documentfoundation.org/contribution/ לקבלת פרטים. כל אחד יכול לתרום בדרך כלשהי.

----------------------------------------------------------------------
הערות על התקנה
----------------------------------------------------------------------

LibreOffice זקוק לגרסה עדכנית של Java Runtime Environment (JRE) for full functionality. JRE אינה חלק מחבילת ההתקנה של LibreOffice, יש להתקין אותה בנפרד.

דרישות מערכת:‏
----------------------------------------------------------------------

* Microsoft Windows XP SP3, Vista, 7, 8, או 10

לידיעתך, לתהליך ההתקנה דרושות הרשאות מנהל.

רישום LibreOffice כתוכנית בררת המחדל לפתיחת קבצים בפורמטי Microsoft Office, אפשר לכפות על ידי הפעלת תוכנית ההתקנה משורת הפקודה עם תוספת הפרמטר הבא:

* REGISTER_ALL_MSO_TYPES=1 יאלץ את הרישום של LibreOffice כאפליקצית בררת המחדל עבור פורמטי Microsoft Office.
* REGISTER_NO_MSO_TYPES=1 יוותר על הרישום של LibreOffice כאפליקצית בררת המחדל עבור פורמטי Microsoft Office.

נא לוודא שיש לך די זיכרון פנוי בתיקייה הזמנים או במערכת שלך, כמוכן יש לוודא שלמשתמש שלך יש לך גישה לקרוא, לכתוב ולהריץ במיקומים הדרושים. יש לסגור את כל שאר היישומים בטרם הפעלת תהליך ההתקנה.

התקנת LibreOffice על מערכות לינוקס מבוססות דביאן/אובונטו
----------------------------------------------------------------------

For instructions on how to install a language pack (after having installed the US English version of LibreOffice), please read the section below entitled Installing a Language Pack.

בעת חילוץ הארכיון שהורדת, ניתן לראות שהתוכן נפרס לתוך תת־תיקייה. יש לפתוח את חלון מנהל הקבצים ולעבור לתיקייה המתחילה ב־„LibreOffice_‎“ שלאחריה מספר הגרסה ופרטים כלשהם על הפלטפורמה.

תיקייה זו מכילה תת־תיקייה בשם "DEBS". יש לעבור לתיקייה "DEBS".

יש ללחוץ על התיקייה בלחצן ימני ולבחור ב־"פתיחה במסוף". ייפתח חלון מסוף. משורת הפקודה של חלון המסוף, יש להזין את הפקודה הבאה (תופיע בקשה להזין את ססמת המשתמש לפני שהפקודה תופעל):

הפקודות הבאות תתקנה את LibreOffice וחבילות השילוב בשולחן העבודה (ניתן להעתיק ולהדביק את הפקודות לתוך המסוף במקום להקליד אותן):

sudo dpkg -i *.deb

תהליך ההתקנה מושלם כעת ואמורים להופיע סמלים ליישום LibreOffice בתפריט יישומים/משרד בשולחן העבודה שלך.

התקנת LibreOffice על גבי Fedora,‏ Suse,‏ Mandriva והפצות לינוקס אחרות באמצעות חבילות RPM
----------------------------------------------------------------------

For instructions on how to install a language pack (after having installed the US English version of LibreOffice), please read the section below entitled Installing a Language Pack.

בעת חילוץ הארכיון שהורדת, ניתן לראות שהתוכן נפרס לתוך תת־תיקייה. יש לפתוח את חלון מנהל הקבצים ולעבור לתיקייה המתחילה ב־„LibreOffice_‎“ ולאחר מכן מספר הגרסה ופרטים כלשהם על הפלטפורמה.

תיקייה זו מכילה תת־תיקייה בשם "RPMS". יש לעבור לתיקייה "RPMS".

יש ללחוץ על התיקייה בלחצן ימני ולבחור ב־"פתיחה במסוף". ייפתח חלון מסוף. משורת הפקודה של חלון המסוף, יש להזין את הפקודה הבאה (תופיע בקשה להזין את ססמת המשתמש לפני שהפקודה תופעל):

למערכות מבוססות פדורה: su -c 'yum install *.rpm'‎

למערכות מבוססות Mandriva‏: sudo urpmi *.rpm

למערכות אחרות מבוססות RPM‏ (openSUSE וכו׳): rpm -Uvh *.rpm

תהליך ההתקנה מושלם כעת ואמורים להופיע סמלים ליישום LibreOffice בתפריט יישומים/משרד בשולחן העבודה שלך.

Alternatively, you can use the 'install' script, located in the toplevel directory of this archive to perform an installation as a user. The script will set up LibreOffice to have its own profile for this installation, separated from your normal LibreOffice profile. Note that this will not install the system integration parts such as desktop menu items and desktop MIME type registrations.

הערות באשר לשילוב בשולחן העבודה בהפצות לינוקס אינן מכוסות בהנחיות ההתקנה שלהלן
----------------------------------------------------------------------

פעולת ההתקנה של LibreOffice על מערכות לינוקס נוספות שלא כוסו בהנחיות התקנה אלו אמורה להיות פשוטה למדי. ההיבט הראשי שבו עלולים להיות שינויים הוא השילוב בשולחן העבודה.

The RPMS (or DEBS, respectively) directory also contains a package named libreoffice5.4-freedesktop-menus-5.4.0.1-1.noarch.rpm (or libreoffice5.4-debian-menus_5.4.0.1-1_all.deb, respectively, or similar). This is a package for all Linux distributions that support the Freedesktop.org specifications/recommendations (http://en.wikipedia.org/wiki/Freedesktop.org), and is provided for installation on other Linux distributions not covered in the aforementioned instructions.

התקנת חבילת שפה
----------------------------------------------------------------------

באפשרותך להוריד את חבילת השפה עבור מערכת ההפעלה והשפה המועדפות עליך. אלו זמינות מאותו המיקום כמו ארכיון ההתקנה הראשי. ממנהל הקבצים Nautilus, יש לחלץ את הארכיון שהתקבל לתוך תיקייה (שלוחן העבודה שלך, לדוגמה). עליך לוודא שיצאת מכל היישומים של LibreOffice (לרבות הפעלה מהירה/QuickStarter אם הופעל).

יש לעבור לתיקייה אליה חילצת את חבילת השפה שהורדת.

כעת יש לעבור לתיקייה שנוצרה במהלך החילוץ. לדוגמה, לחבילת השפה העברית עבור מערכת דביאן/אובונטו בגרסת 32 סיביות, שם התיקייה יתחיל ב־LibreOffice_‎, לאחר מכן מספר הגרסה ובהמשך Linux_x86_langpack-deb_he.

כעת עליך לעבור לתיקייה המכילה את החבילות להתקנה. במערכות המבוססות על דביאן/אובונטו, שם התיקייה הוא DEBS. במערכות פדורה, OpenSUSE או Mandriva שם התיקייה הוא RPMS.

ממנהל הקבצים Nautilus יש ללחוץ בלחצן ימני על התיקייה ולבחור בפקודה "פתיחה במסוף". בחלון המסוף שנפתח, יש להריץ את הפקודה להתקנת חבילת השפה (עם כל הפקודות שלהלן, יתכן שתופיע בקשה להזנת ססמת המשתמש שלך):

למערכות מבוססות דביאן/אובונטו: sudo dpkg -i *.deb

למערכות מבוססות פדורה: su -c 'yum install *.rpm'‎

למערכות מבוססות Mandriva‏: sudo urpmi *.rpm

למערכות אחרות המשתמשות ב־RPM‏ (OpenSUSE וכו׳): rpm -Uvh *.rpm

כעת יש להפעיל את אחד מיישומי LibreOffice‏ – Writer לדוגמה. יש ללחוץ על התפריט Tools ולבחור ב־Options. בתיבת הדו־שיח Options יש ללחוץ על "Language Settings" ואז ללחוץ על "Languages". כעת יש לגלול כלפי מטה ברשימה "User Interface" ולבחור בשפה שהרגע התקנת. עליך לעשות את אותו הדבר עבור "Locale setting", "Default currency" ו־"Default languages for documents".

לאחר התאמת הגדרות אלו, יש ללחוץ על אישור. תיבת הדו־שיח תיסגר ותופיע בפניך הודעה שתאמר לך שהשינויים שלך יופעלו רק לאחר היציאה מ־LibreOffice והפעלה מחדש (יש לזכור לצאת גם מ"הפעלה מהירה"/QuickStarter אם הוא פעיל).

בפעם הבאה ש־LibreOffice תופעל, היא תופעל בשפה שזה עתה התקנת.

תקלות במהלך הפעלת התכנית
----------------------------------------------------------------------

אירעו תקלות בעת הפעלת LibreOffice (לדוגמה: יישומים קופאים) כמו גם תקלות בתצוגה שנגרמות על ידי כרטיס המסך. אם בעיות אלו מתרחשות, נא לעדכן את מנהל ההתקן של כרטיס המסך או לנסות להשתמש במנהל ההתקן שסופק עם מערכת ההפעלה שלך. תקלות בהצגת עצמים תלת־ממדיים ניתנות לפתרון על ידי נטרול האפשרות „שימוש ב־OpenGL“ תחת ‚כלים - אפשרויות - LibreOffice - תצוגה - תצוגת תלת־ממד‘.

----------------------------------------------------------------------
ALPS/Synaptics notebook touchpads בחלונות של מיקרוסופט
----------------------------------------------------------------------

עקב בעיה במנהל התקן של Windows, לא ניתן לגלול במסמכי LibreOffice כאשר מחליקים את האצבע לאורך ALPS/Synaptics touchpad.

כדי לאפשר גלילה באמצעות touchpad, יש להוסיף את השורות הבאות לקובץ ההגדרות "C:\Program Files\Synaptics\SynTP\SynTPEnh.ini" ולאתחל מחדש את המחשב:

[LibreOffice]

FC = "SALFRAME"

SF = 0x10000000

SF |= 0x00004000

הערה: מיקום קובץ התצורה עלול להשתנות בגסאות שונות של חלונות.‏

----------------------------------------------------------------------
מקשי קיצור
----------------------------------------------------------------------

Only shortcut keys (key combinations) not used by the operating system can be used in LibreOffice. If a key combination in LibreOffice does not work as described in the LibreOffice Help, check if that shortcut is already used by the operating system. To rectify such conflicts, you can change the keys assigned by your operating system. Alternatively, you can change almost any key assignment in LibreOffice. For more information on this topic, refer to the LibreOffice Help or the Help documentation of your operating system.
