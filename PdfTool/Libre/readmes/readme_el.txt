Για τις τελευταίες ενημερώσεις για αυτό το αρχείο διάβασε με, δείτε http://www.libreoffice.org/welcome/readme.html

Αυτό το αρχείο περιέχει ενδιαφέρουσες πληροφορίες για το λογισμικό LibreOffice. Συνιστάται η πολύ προσεκτική ανάγνωση πριν την έναρξη της εγκατάστασης.

Η κοινότητα του LibreOffice είναι υπεύθυνη για την ανάπτυξη αυτού του προϊόντος και σας προσκαλεί να συμμετάσχετε ως μέλος της κοινότητας. Εάν είστε νέος χρήστης, μπορείτε να επισκεφτείτε τον τόπο LibreOffice, όπου θα βρείτε πολλές πληροφορίες για το σχέδιο LibreOffice και τις κοινότητες που υπάρχουν γύρω του. Μεταβείτε στο http://www.libreoffice.org/.

Είναι το LibreOffice πραγματικά ελεύθερο για κάθε χρήστη;
----------------------------------------------------------------------

Το LibreOffice είναι ελεύθερο για χρήση από οιονδήποτε. Μπορείτε να πάρετε αυτό το αντίγραφο του LibreOffice και να το εγκαταστήσετε σε όσους υπολογιστές θέλετε και να το χρησιμοποιήσετε για κάθε επιθυμητό σκοπό (συμπεριλαμβάνοντας εμπορική, κυβερνητική, δημόσιας διοίκησης και εκπαιδευτική χρήση). Για περισσότερες λεπτομέρειες δείτε το ενσωματωμένο κείμενο άδειας μαζί με αυτή τη λήψη του LibreOffice.

Γιατί το LibreOffice είναι ελεύθερο για κάθε χρήστη;
----------------------------------------------------------------------

Μπορείτε να χρησιμοποιήσετε αυτό το αντίγραφο του LibreOffice χωρίς χρέωση επειδή ατομικοί συντελεστές και συλλογικοί χορηγοί σχεδίασαν, ανέπτυξαν, έλεγξαν μετέφρασαν, τεκμηρίωσαν, υποστήριξαν, εμπορεύτηκαν και βοήθησαν με πολλούς άλλους τρόπους να κάνουν το LibreOffice αυτό που είναι σήμερα - το παγκόσμια κορυφαίο παραγωγικό ανοιχτό λογισμικό για σπίτια και γραφεία.

Εάν εκτιμάτε τις προσπάθειες τους και θα θέλατε να επιβεβαιώσετε ότι το LibreOffice θα συνεχίσει να διατίθεται για πολύ στο μέλλον, παρακαλώ σκεφτείτε τη συμμετοχή σας στο σχέδιο - δείτε http://www.documentfoundation.org/contribution/ για λεπτομέρειες. Ο καθένας μπορεί να συμβάλει με κάποιον τρόπο.

Σημειώσεις εγκατάστασης
----------------------------------------------------------------------

Το LibreOffice απαιτεί μια πρόσφατη έκδοση του περιβάλλοντος χρόνου εκτέλεσης Java (JRE) για πλήρη λειτουργικότητα. Το JRE δεν είναι μέρος του πακέτου εγκατάστασης LibreOffice, πρέπει να εγκατασταθεί ξεχωριστά.

Απαιτήσεις συστήματος
----------------------------------------------------------------------

* Microsoft Windows XP SP3, Vista, 7, 8, ή 10

Παρακαλώ λάβετε υπόψη ότι απαιτούνται δικαιώματα διαχειριστή για τη διαδικασία εγκατάστασης.

Η εγγραφή του LibreOffice ως προεπιλεγμένη εφαρμογή για τύπους αρχείων Microsoft Office μπορεί να εξαναγκαστεί ή να καταργηθεί χρησιμοποιώντας την ακόλουθη γραμμή εντολής που αλλάζει με την εγκατάσταση:

* Το REGISTER_ALL_MSO_TYPES=1 θα εξαναγκάσει την καταχώριση του LibreOffice ως προεπιλεγμένης εφαρμογής στις μορφές Microsoft Office.
* Το REGISTER_NO_MSO_TYPES=1 θα παρεμποδίσει την καταχώριση του LibreOffice ως προεπιλεγμένης εφαρμογής για τις μορφές του Microsoft Office.

Παρακαλώ, βεβαιωθείτε ότι έχετε αρκετή ελεύθερη μνήμη στον προσωρινό κατάλογο του συστήματος σας και βεβαιωθείτε ότι έχετε τα δικαιώματα πρόσβασης ανάγνωσης, εγγραφής και εκτέλεσης. Κλείστε όλα τα άλλα προγράμματα πριν ξεκινήσετε τη διαδικασία εγκατάστασης.

Εγκατάσταση του LibreOffice σε συστήματα Linux που βασίζονται σε Debian/Ubuntu
----------------------------------------------------------------------

Για οδηγίες εγκατάστασης ενός πακέτου γλώσσας (αφού έχετε εγκαταστήσει την έκδοση Αγγλικά-ΗΠΑ του LibreOffice), παρακαλούμε διαβάστε την παρακάτω ενότητα που τιτλοφορείται εγκατάσταση πακέτου γλώσσας.

Όταν αποσυμπιέζετε την αρχειοθήκη που μεταφορτώσατε, θα δείτε ότι τα περιεχόμενα έχουν αποσυμπιεστεί σε έναν υποκατάλογο. Ανοίξτε ένα παράθυρο διαχειριστή αρχείων και αλλάξτε τον κατάλογο σε εκείνον που αρχίζει με "LibreOffice_", ακολουθούμενο από τον αριθμό έκδοσης και κάποιες πληροφορίες λειτουργικού.

Αυτός ο κατάλογος περιέχει έναν υποκατάλογο με όνομα "DEBS". Αλλάξτε τον κατάλογο στον κατάλογο "DEBS".

Δεξί πάτημα μες τον κατάλογο και επιλέξτε "άνοιγμα σε τερματικό". Ένα παράθυρο τερματικού θα ανοίξει. Από τη γραμμή εντολών του παραθύρου τερματικού, εισάγετε την παρακάτω εντολή (θα σας ζητηθεί να εισάγετε τον κωδικό πρόσβασης του υπερχρήστη πριν την εκτέλεση της εντολής):

Οι παρακάτω εντολές θα εγκαταστήσουν το LibreOffice και τα πακέτα ολοκλήρωσης της επιφάνειας εργασίας (μπορείτε να αντιγράψετε απλά και να τις επικολλήσετε στην οθόνη τερματικού αντί να προσπαθήσετε να τις πληκτρολογήσετε):

sudo dpkg -i *.deb

Η διαδικασία εγκατάστασης ολοκληρώθηκε τώρα και πρέπει να έχετε εικονίδια για όλες τις εφαρμογές του LibreOffice στο μενού της επιφάνειας εργασίας σας προγράμματα/γραφείο.

Εγκατάσταση του LibreOffice στα Fedora, openSUSE, Mandriva και άλλα συστήματα Λίνουξ χρησιμοποιώντας πακέτα RPM
----------------------------------------------------------------------

Για οδηγίες εγκατάστασης ενός πακέτου γλώσσας (αφού έχετε εγκαταστήσει την έκδοση Αγγλικά-ΗΠΑ του LibreOffice), παρακαλούμε διαβάστε την παρακάτω ενότητα που τιτλοφορείται εγκατάσταση πακέτου γλώσσας.

Όταν αποσυμπιέζετε την αρχειοθήκη που μεταφορτώσατε, θα δείτε ότι τα περιεχόμενα έχουν αποσυμπιεστεί σε έναν υποκατάλογο. Ανοίξτε ένα παράθυρο διαχειριστή αρχείων και αλλάξτε τον κατάλογο σε εκείνον που αρχίζει με "LibreOffice_", ακολουθούμενο από τον αριθμό έκδοσης και κάποιες πληροφορίες λειτουργικού.

Αυτός ο κατάλογος περιέχει έναν υποκατάλογο με όνομα "RPMS". Αλλάξτε κατάλογο στον κατάλογο "RPMS".

Δεξί πάτημα μες τον κατάλογο και επιλέξτε "άνοιγμα σε τερματικό". Ένα παράθυρο τερματικού θα ανοίξει. Από τη γραμμή εντολών του παραθύρου τερματικού, εισάγετε την παρακάτω εντολή (θα σας ζητηθεί να εισάγετε τον κωδικό πρόσβασης του υπερχρήστη πριν την εκτέλεση της εντολής):

Για συστήματα βασισμένα σε Fedora: su -c 'yum install *.rpm'

Για συστήματα βασισμένα σε Mandriva: sudo urpmi *.rpm

Για άλλα συστήματα με βάση RPM (openSUSE, κλπ): rpm -Uvh *.rpm

Η διαδικασία εγκατάστασης ολοκληρώθηκε τώρα και πρέπει να έχετε εικονίδια για όλες τις εφαρμογές του LibreOffice στο μενού της επιφάνειας εργασίας σας προγράμματα/γραφείο.

Εναλλακτικά, μπορείτε να χρησιμοποιήσετε το σενάριο 'install', που βρίσκεται στον κατάλογο ανωτάτου επιπέδου αυτής της αρχειοθήκης για να εκτελέσει μια εγκατάσταση ως χρήστης. Το σενάριο θα ρυθμίσει το LibreOffice να έχει την δικιά του κατατομή για αυτήν την εγκατάσταση, χωριζόμενο από τη δικό σας κανονική κατατομή LibreOffice. Σημειώστε ότι αυτό δεν θα εγκαταστήσει τα μέρη ολοκλήρωσης συστήματος όπως τα στοιχεία μενού επιφάνειας εργασίας και εγγραφές τύπου MIME της επιφάνειας εργασίας.

Σημειώσεις που αφορούν την ολοκλήρωση της επιφάνειας εργασίας για διανομές Λίνουξ που δεν καλύπτονται από τις παραπάνω οδηγίες εγκατάστασης
----------------------------------------------------------------------

Θα πρέπει να είναι εύκολα δυνατό να εγκαταστήσετε το LibreOffice σε άλλες διανομές Λίνουξ που δεν καλύπτονται ειδικά από αυτές τις οδηγίες εγκατάστασης. Η κύρια όψη για την οποία μπορούν να αντιμετωπιστούν διαφορές είναι η ολοκλήρωση της επιφάνειας εργασίας.

Ο κατάλογος RPMS (ή DEBS, αντίστοιχα) μπορεί να περιέχει επίσης ένα πακέτο με όνομα libreoffice5.4-freedesktop-menus-5.4.0.1-1.noarch.rpm (ή libreoffice5.4-debian-menus_5.4.0.1-1_all.deb, αντίστοιχα ή κάτι παρόμοιο). Αυτό είναι ένα πακέτο για όλες τις διανομές Linux που υποστηρίζουν τις προδιαγραφές/συστάσεις του Freedesktop.org (http://en.wikipedia.org/wiki/Freedesktop.org) και παρέχονται για εγκατάσταση σε άλλες διανομές Λίνουξ που δεν καλύπτονται από τις προμνημονευθείσες οδηγίες.

Εγκατάσταση γλωσσικού πακέτου
----------------------------------------------------------------------

Κατεβάστε το επιθυμητό γλωσσικό πακέτο για το λειτουργικό σας. Είναι διαθέσιμα από την ίδια τοποθεσία όπως και η αρχειοθήκη της κύριας εγκατάστασης. Από το διαχειριστή αρχείων Ναυτίλου εξάγεται την κατεβασμένη αρχειοθήκη σε έναν κατάλογο (την επιφάνεια εργασίας, για παράδειγμα). Βεβαιωθείτε ότι έχετε βγει από όλες τις εφαρμογές του LibreOffice (συμπεριλαμβανόμενης της γρήγορης εκκίνησης, εάν έχει ξεκινήσει).

Πηγαίνετε στον κατάλογο όπου έχετε εξαγάγει το κατεβασμένο γλωσσικό πακέτο.

Τώρα αλλάξτε τον κατάλογο στον κατάλογο που δημιουργήθηκε κατά τη διεργασία εξαγωγής. Για παράδειγμα, για το γαλλικό πακέτο γλώσσας για ένα σύστημα 32 δυαδικών με βάση Ντέμπιαν/Ουμπούντου, ο κατάλογος ονομάζεται LibreOffice_, συν κάποια πληροφορία έκδοσης, συν Linux_x86_langpack-deb_fr.

Τώρα αλλάξτε κατάλογο στον κατάλογο που περιέχει τα πακέτα για εγκατάσταση. Σε συστήματα με βάση Ντέμπιαν/Ουμπούντου, ο κατάλογος θα είναι DEBS. Σε συστήματα Fedora, openSUSE ή Mandriva, ο κατάλογος θα είναι RPMS.

Από το διαχειριστή αρχείων Ναυτίλο, δεξί πάτημα στον κατάλογο και επιλογή της εντολής "άνοιγμα σε τερματικό". Στο παράθυρο τερματικού που μόλις ανοίξατε, εκτελέστε την εντολή για εγκατάσταση του γλωσσικού πακέτου (με όλες τις παρακάτω εντολές, ίσως σας ζητηθεί να εισάγετε τον κωδικό πρόσβασης διαχειριστή):

Για συστήματα βασισμένα σε Debian/Ubuntu: sudo dpkg -i *.deb

Για συστήματα βασισμένα σε Fedora: su -c 'yum install *.rpm'

Για συστήματα βασισμένα σε Mandriva: sudo urpmi *.rpm

Για άλλα συστήματα που χρησιμοποιούν RPM (openSUSE, κλπ.): rpm -Uvh *.rpm

Τώρα ξεκινήστε μια από τις εφαρμογές του LibreOffice - παραδείγματος χάρη, Writer. Μετάβαση στο μενού εργαλείων και διαλέξτε Επιλογές. Στο πλαίσιο διαλόγου Επιλογές, πατήστε στο "Ρυθμίσεις γλώσσας" και έπειτα πατήστε "Γλώσσες". Ανοίξτε τη λίστα "Διεπαφή χρήστη" και διαλέξτε τη γλώσσα που μόλις εγκαταστήσατε. Εάν θέλετε, κάντε το ίδιο πράγμα για την "τοπική ρύθμιση", το "προεπιλεγμένο νόμισμα" και την "προεπιλεγμένη γλώσσα για έγγραφα".

Μετά την προσαρμογή αυτών των ρυθμίσεων, πατήστε εντάξει. Το πλαίσιο διαλόγου θα κλείσει και θα δείτε ένα μήνυμα πληροφορίας που σας λέει ότι οι αλλαγές σας θα ενεργοποιηθούν μόνο μετά την έξοδο του LibreOffice και επανεκκίνηση του (θυμηθείτε να βγείτε επίσης από την γρήγορη εκκίνηση εάν έχει ξεκινήσει).

Την επόμενη φορά που θα ξεκινήσετε το LibreOffice, θα ξεκινήσει στη γλώσσα που μόλις εγκαταστήσατε.

Προβλήματα κατά την εκκίνηση του προγράμματος
----------------------------------------------------------------------

Δυσκολίες εκκίνησης του LibreOffice (π.χ. οι εφαρμογές κολλάνε), όπως και προβλήματα με την εμφάνιση της οθόνης συχνά οφείλονται στον οδηγό της κάρτας γραφικών σας. Αν σας παρουσιάζονται αυτά τα προβλήματα ενημερώστε τον οδηγό της κάρτας γραφικών σας ή χρησιμοποιήστε αυτόν που έχει το λειτουργικό σας σύστημα. Δυσκολίες εμφάνισης 3Δ αντικειμένων μπορούν να επιλυθούν με την απενεργοποίηση της επιλογής "Χρήση OpenGL" στο 'Εργαλεία - Επιλογές - LibreOffice - Προβολή - Προβολή 3Δ'.

----------------------------------------------------------------------
ALPS/Synaptics για πινακίδα επαφής φορητού στα Windows
----------------------------------------------------------------------

Εξαιτίας ενός προβλήματος οδηγού από τα Windows, δε μπορείτε να κυλήσετε μέσα από τα έγγραφα του LibreOffice, όταν μετακινείτε το δάχτυλό σας κατά μήκος μιας πινακίδας επαφής ALPS/Synaptics.

Για να ενεργοποιήσετε την κύλιση της πινακίδας επαφής, προσθέστε τις ακόλουθες γραμμές στο αρχείο ρυθμίσεων "C:\Program Files\Synaptics\SynTP\SynTPEnh.ini", και κάνετε επανεκκίνηση του υπολογιστή σας:

[LibreOffice]

FC = "SALFRAME"

SF = 0x10000000

SF |= 0x00004000

Η τοποθεσία του αρχείου ρυθμίσεων μπορεί να ποικίλει ανάλογα με τις διαφορετικές εκδόσεις των Windows.

Πλήκτρα συντόμευσης
----------------------------------------------------------------------

Στο LibreOffice μπορούν να χρησιμοποιηθούν μόνο πλήκτρα συντόμευσης (συνδυασμοί πλήκτρων) που δεν χρησιμοποιούνται από το λειτουργικό σύστημα. Αν ένας συνδυασμός πλήκτρων στο LibreOffice δεν ανταποκρίνεται όπως περιγράφεται στη Βοήθεια του LibreOffice, ελέγξτε αν η συντόμευση αυτή χρησιμοποιείται ήδη από το λειτουργικό σύστημα. Για να διορθώσετε τέτοιες συγκρούσεις, μπορείτε να αλλάξετε τα πλήκτρα που έχουν ανατεθεί από το λειτουργικό σύστημα. Εναλλακτικά, μπορείτε να αλλάξετε σχεδόν όλες τις αναθέσεις συντομεύσεων στο LibreOffice. Για περισσότερες πληροφορίες σχετικά με αυτό το θέμα, μεταβείτε στη Βοήθεια του LibreOffice ή την τεκμηρίωση της βοήθειας του λειτουργικού σας συστήματος.

----------------------------------------------------------------------
Προβλήματα κατά την αποστολή εγγράφων ως ηλεκτρονικά μηνύματα με το LibreOffice
----------------------------------------------------------------------

Κατά την αποστολή ενός εγγράφου μέσω των επιλογών 'Αρχείο - Αποστολή - Έγγραφο ως μήνυμα' ή 'Έγγραφο ως συνημμένο PDF' μπορούν να προκύψουν προβλήματα (καταρρεύσεις προγράμματος ή κολλήματα). Αυτό οφείλεται στο αρχείο συστήματος των Windows "Mapi" (Messaging Application Programming Interface) που προκαλεί προβλήματα σε μερικές εκδόσεις αρχείων. Δυστυχώς, το πρόβλημα δεν μπορεί να περιοριστεί σε ένα βέβαιο αριθμό εκδόσεων. Για περισσότερες πληροφορίες επισκεφτείτε τη σελίδα http://www.microsoft.com για να αναζητήσετε στη βάση γνώσεων Microsoft το αρχείο "mapi dll".

Εδώ είναι μερικές λίστες αλληλογραφίας στις οποίες μπορείτε να εγγραφείτε στο http://www.libreoffice.org/contribution/

* Νέα: announce@documentfoundation.org *συνιστάται για όλους τους χρήστες* (μικρή κυκλοφορία)
* Κύρια λίστα χρηστών: users@global.libreoffice.org *εύκολος τρόπος παρακολούθησης συζητήσεων* (υψηλή κίνηση)
* Έργο αγοράς: marketing@global.libreoffice.org *πέρα από την ανάπτυξη* (γίνεται πυκνής κίνησης)
* Λίστα γενικής ανάπτυξης: libreoffice@lists.freedesktop.org (υψηλή κυκλοφορία)

Γίνετε μέλος σε ένα ή περισσότερα έργα
----------------------------------------------------------------------

Μπορείτε να πραγματοποιήσετε μεγαλύτερες συνεισφορές σε αυτό το σημαντικό έργο ανοικτού κώδικα ακόμη και αν περιορίζεστε στη σχεδίαση του λογισμικού ή στη δοκιμή του κώδικα. Ναι, εσείς!

Ελπίζουμε ότι θα σας αρέσει το νέο LibreOffice 5.4 και θα συμμετέχετε και εσείς στην προσπάθεια μας.
